# Backend – Travel Together

Acest folder conține implementarea backend-ului aplicației **Travel Together**, realizată în **PHP** folosind arhitectura **MVC**

## Structură planificată

- **/controllers** – logica aplicației și gestionarea rutelor
- **/models** – clase pentru accesarea bazei de date MySQL
- **/views** – template-uri generate de backend
- **/core** – componente de bază ale framework-ului MVC (router, controller, model)
- **index.php** – punctul de intrare al aplicației

## Funcționalități

- CRUD pentru utilizatori, călătorii și recenzii
- Conectarea aplicației la baza de date MySQL
- Implementarea motorului de recomandări personalizate (content-based)
- Integrarea unui modul AI (chatbot)
- Procesarea preferințelor utilizatorului
- Gestionarea sesiunilor, validări și securitate

Backend-ul va reprezenta nucleul aplicației și va susține toate modulele majore: recomandări, harta interactivă, profilul utilizatorului și chatbot-ul AI.
