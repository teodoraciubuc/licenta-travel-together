# Frontend – Travel Together

Acest folder conține partea de interfață a aplicației

## Frontend-ul este dezvoltat folosind:

- **HTML5** – structurarea paginilor aplicației
- **CSS3** – stilizarea elementelor
- **Bootstrap 5** – layout responsive și componente UI
- **JavaScript** – funcționalități dinamice și interacțiune cu backend-ul
- **Leaflet.js** – afișarea hărții interactive
- **OpenStreetMap** – furnizor de hărți utilizat în proiect

## Structură planificată

- **/assets/css** – stiluri personalizate
- **/assets/js** – scripturi pentru logica front-end
- **/assets/img** – imagini utilizate în aplicație
- **/views** – pagini responsabile pentru prezentarea datelor venite din backend

## Funcționalități

- Interfața pentru vizualizarea recomandărilor personalizate
- Afișarea hărții interactive cu destinații recomandate sau vizitate
- Formulare pentru gestionarea preferințelor utilizatorului
- Interfață pentru chatbot-ul AI
- Paginile aplicației (login, profil, hartă, recomandări)

Frontend-ul va comunica permanent cu backend-ul pentru a afișa datele relevante utilizatorului.
